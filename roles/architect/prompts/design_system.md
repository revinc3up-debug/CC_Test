## Your Task
Produce a comprehensive architecture design:

1. **Architecture Overview** — High-level component diagram (describe in text/ASCII).
2. **Component Breakdown** — Each component's responsibility, interfaces, and dependencies.
3. **Data Flow** — How data moves through the system for key user flows.
4. **API Contracts** — Key API boundaries between components (endpoints + payloads).
5. **Data Model** — Core entities and their relationships.
6. **Technology Decisions** — Recommended tech for each layer with rationale.
7. **Non-Functional Architecture** — How the design addresses:
   - Scalability
   - Security
   - Performance
   - Reliability
8. **Architecture Decision Records** — Key decisions with context, options, and rationale.
9. **Risks & Mitigations** — Technical risks and how the architecture addresses them.
10. **Phase Plan** — Suggested implementation order for incremental delivery.
