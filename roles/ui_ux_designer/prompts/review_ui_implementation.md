## Review Criteria
1. **Visual Accuracy** — Spacing, colors, typography match the spec.
2. **Interaction Fidelity** — Animations, transitions, hover/focus states.
3. **Responsive Behavior** — Correct layout at all breakpoints.
4. **Accessibility** — Keyboard navigation, screen reader, contrast.
5. **Edge Cases** — Long text, empty states, loading states, error states.
6. **Consistency** — Adherence to the design system tokens and patterns.

Provide specific feedback with severity (must-fix / should-fix / nice-to-have).
